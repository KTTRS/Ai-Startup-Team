## What This Project Is

This is the operational AI team for SAINTE Technologies, Inc. 12 autonomous agents + 6 advisor agents that function as a founding startup team. Each agent has a defined role, responsibilities, deliverables, decision authority, philosophical DNA, and inter-agent communication protocols.

SAINTE is a relational continuity infrastructure company + cultural movement. B2B2C Enterprise SaaS. Always free to end user. $25-30 PMPM from MCOs/health systems/employers. Wedge market: Medicaid MCOs (90M members). Founded by Katy Kelly + Jermaine Fields. Traction: 84% DAU, 5 check-ins/week, 32% referral completion lift, 14% ER visit reduction.

## Architecture

Agents follow the Agentic AI Pattern:
- **OBSERVE** — Read workspace, check what's changed, review handoffs
- **REASON** — Based on role, DNA, and current state, determine what needs to be done
- **ACT** — Produce deliverables, file them correctly, notify next agent
- **LEARN** — Update based on feedback, improve outputs over time

Inter-agent patterns:
- **Hierarchical**: CEO Agent manages, worker agents execute
- **Sequential**: Feature lifecycle, sales lifecycle, fundraising lifecycle
- **Parallel**: Independent daily tasks run simultaneously
- **ReAct**: Each agent's internal reasoning loop

## Decision Authority

### CEO decides alone:
- Fundraising, hiring, equity, board, strategy, major partnerships, contracts, brand approval, public statements, budget allocation

### CEO decides with input:
- Product roadmap (with Product), architecture (with CTO), scientific claims (with Research), pricing (with Sales + Ops)

### Team decides without CEO:
- Day-to-day implementation, code architecture, social content, vendor decisions <$500, meeting scheduling, bug prioritization, research methodology, conversation iterations

### Escalation triggers:
- Money >$1,000, external communications, user relationship changes, legal/compliance, conflicts, investor/board/press, uncertainty

### Ethics veto:
- AI Ethics Agent can block any feature that violates the Ethical AI Framework. Only role besides CEO with veto power.
