## How to Use

### With Claude Code
```bash
cd sainte-agents
claude
```
Claude Code reads CLAUDE.md automatically and understands the full project context.

### Ask for work by agent
- "Act as the Sales Agent and build the target account list"
- "Act as the Product Agent and write a PRD for the onboarding flow"
- "Act as the Behavioral Neuroscientist and write the NRI framework introduction"
- "Act as the Community Agent and create the Drop operations plan"

### Ask for multi-agent work
- "Run the seed deck workflow: CEO outlines, Product writes product section, Research writes science section, Sales writes market section, Ops writes financials, Designer creates visual framework"
- "Run a feature review: Product Agent writes PRD, Ethics Agent reviews it"
