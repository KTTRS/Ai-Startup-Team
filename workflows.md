# Core Workflows

## Feature Lifecycle
1. Idea (Anyone) → Problem statement
2. Research Input (Behavioral Neuroscientist) → Research brief
3. PRD (Product Agent) → Full PRD
4. Ethical Review (Ethics Agent) → Passed or blocked
5. Technical Scoping (CTO) → Feasibility + timeline
6. Conversation Design (Product Agent) → Specs, prompts
7. Build (CTO + Engineers) → Code in staging
8. Testing (All) → Bugs fixed
9. Release (CTO) → Production
10. Measure (Product) → Metrics 1wk, 1mo

## Sales Lifecycle
Prospect → Research → Outreach → Discovery → Presentation → Proposal → Negotiation → Close → Implementation

## Seed Deck Workflow
CEO outlines → Product (product section) → Research (science) → Sales (market) → Ops (financials) → Brand (design) → CEO synthesizes → Ethics reviews → FQHC Advisor validates

## Content Lifecycle
Concept → Brief → Create → Review → Publish → Engage

## Fundraising Lifecycle
Strategy → Materials → Target List → Outreach → Meetings → Diligence → Term Sheet → Close → Post-Close
